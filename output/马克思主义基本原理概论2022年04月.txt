马克思主义基本原理概论2022年04月（选择题）

马克思主义哲学的直接理论来源是（  ）（B）
A 古希腊罗马哲学
B 德国古典哲学
C 近代英国哲学
D 近代法国哲学
解析：解析：马克思主义哲学的直接理论来源德国古典哲学，德国古典哲学代表人物是黑格尔和费尔巴哈，马克思主义哲学在德国古典哲学的基础上，提出了辩证唯物主义和历史唯物主义。

学习马克思主义的根本方法是（  ）（A）
A 理论联系实际
B 更新思想现念
C 熟读经典著作
D 了解党的文献
解析：解析：理论联系实际是学习马克思主义的根本方法。

下列各项属于一元论哲学思想的是（  ）（B）
A 世界万物是普遍联系的
B 世界万物有一个共同的本质
C 世界万物是可以认识的
D 世界万物有多种多样的形态
解析：解析：一元论承认世界是统一的，认为世界上的万事万物有一个共同的本质或本原。

实践的基本形式主要有三种。下列各项属于处理社会关系的实践形式的是（  ）（C）
A 种地做工
B 教学科研
C 社会改革
D 治病救人
解析：解析：实践的三种基本形式：物质生产实践（最基本实践活动）；处理社会关系的实践，如社会管理、阶级斗争、社会革命、社会改革；科学实验。

唯物辩证法是关于自然、社会和思维联系与发展一般规律的科学基中，否定之否定规律揭示的是事物发展的（  ）（D）
A 前提和条件
B 状态和形式
C 动力和源泉
D 趋势和道路
解析：解析：对立统一规律揭示了事物发展的动力和源泉。质量互变规律揭示了事物发展的状态和形式。否定之否定规律揭示了事物发展的趋势和道路。

抹杀意识的客观性，把意识说成是纯粹主观自生的，这是一种（  ）（A）
A 唯心主义的观点
B 庸俗唯物主义的观点
C 形而上学的观点
D 不可知论的观点
解析：解析：意识的本质：（1）意识是人脑的机能，人脑是意识的物质器官。（2）意识是客观世界的主观映象，是人脑对客观世界的反映。意识是主观和客观的统一。如果抹杀意识的主观特征，把意识等同于物质或客观事物，把意识说成是人脑的分泌物，就是一种庸俗唯物主义观点。如果抹杀意识的客观性，把意识说成纯粹是主观自生的，就是一种唯心主义观点。（3）意识是社会的产物。

人们追求符合客观实际的科学认识，归根结底是为了（  ）（D）
A 透过现象揭示本质
B 满足所有人的精神追求
C C通过偶然发现必然
D 实现对客观世界的改造
解析：解析：人们追求符合客观实际的科学认识，归根结底是为了实现对客观世界的改造。

理性认识的三种形式是（  ）（B）
A 感觉、概念、判断
B 概念、判断、推理
C 感觉、知觉、表象
D 感觉、知觉、推理
解析：解析：感性认识是认识的初级阶段，包括感觉、知觉和表象，是反映事物的片面、现象、外部联系；理性认识是认识的高级阶段，包括概念、判断和推理（低级到高级），是反映事物的全体和本质。

下列各项正确说明真理问题的是（  ）（D）
A 真理是“绝对理念”的自我显现
B 真理是观念与主体的感觉相符合
C 真理是对客观事物及其规律的能动反映
D 真理是对客观事物及其规律的正确反映
解析：解析：真理是标志主观同客观相符合的哲学范畴，是正确地反映客体的认识。认识和对象相一致、相符合，这是真理最根本的规定性。认识具有真理性，就是说认识与客观实际相符合。

对社会存在和社会意识关系问题的不同回答是（  ）（C）
A 划分唯物主义和唯心主义的标准
B 划分可知论和不可知论的标准
C 划分唯物史观和唯心史观的标准
D 划分辩证法和形而上学的标准
解析：解析：对社会存在和社会意识关系问题的不同回答是划分唯物史观和唯心史观的标准。

衡量生产力水平的客观尺度是（  ）（A）
A 生产工具的状况
B 劳动对象的广度
C 劳动产品的数量
D 劳动产品的质量
解析：解析：生产工具是劳动资料的主要内容，是生产力发展水平的标志。

下列各项关于社会改革的说法中，正确的是（  ）（D）
A 社会改革仅仅是经济领域的变革
B 社会改革仅仅是社会心态的改良
C 社会改革是制度体制的根本改变
D 社会改革是社会体制的改善和革新
解析：解析：社会改革的实质：社会改革是在一定社会制度下，为了解决生产关系不适合生产力、上层建筑不适合和经济基础的某些部分或环节，使该社会制度得到持续存在与发展或自我完善，而对社会体制进行的改善与革新。

区分普通人物和历史人物，主要是看他们（  ）（B）
A 社会财富的多寡
B 历史作用的大小
C 文化水平的高低
D 社会贡献的有无
解析：解析：区分普通人物和历史人物，主要是看他们历史作用的大小。

商品生产者同一劳动过程不可分割的两个方面是（  ）（D）
A 简单劳动和复杂劳动
B 私人劳动和社会劳动
C 必要劳动和剩余劳动
D 具体劳动和抽象劳动
解析：解析：商品生产者同一劳动过程不可分割的两个方面是具体劳动和抽象劳动。

以下关于商品使用价值的表述中不正确的是（  ）（C）
A 使用价值是价值的物质承担者
B 使用价值是商品的自然属性
C 同一物品只能有一种使用价值
D 没有使用价值的物品不可能有价值
解析：解析：商品使用价值的含义：一种物品能够满足人们某种需要的属性，即物品的有用性。商品使用价值的性质：使用价值是商品的自然属性，它是由物品的物理、化学、生物等特性所决定的。商品必然同时具有使用价值和价值两个因素。没有使用价值的物品，不可能有价值，使用价值是价值的物质承担者，而价值则寓于使用价值之中；有使用价值但没有价值的物品，也不是商品，价值是商品的最本质因素。

货币转化为资本的前提是（  ）（C）
A 资本完成原始积累
B 商品生产者出现两极分化
C 劳动力成为商品
D 劳动者有人身自由
解析：解析：劳动力成为商品，是货币转化为资本的前提。

必要劳动时间不变，通过工作日的延长而生产的剩余价值是（  ）（A）
A 绝对剩余价值
B 相对剩余价值
C 超额剩余价值
D 平均剩余价值
解析：解析：在雇佣工人的必要劳动时间不变的条件下，由于工作日的绝对延长而生产的剩余价值，叫作绝对剩余价值。工作日长度不变的条件下，由于缩短必要劳动时间、相应延长剩余劳动时间而生产的剩余价值，叫作相对剩余价值。超额剩余价值就是商品的个别价值低于社会价值的差额。

马克思研究资本主义经济，写作《资本论》的时代，主要资本主义国家处于（  ）（A）
A 自由竞争资本主义阶段
B 一般垄断资本主义阶段
C 私人垄断资本主义阶段
D 国家垄断资本主义阶段
解析：解析：马克思研究资本主义经济，写作《资本论》的时代，主要资本主义国家处于自由竞争资本主义阶段。

各种垄断组织形式中，垄断了某种商品生产和经营全过程的独立的大型垄断企业是（  ）（C）
A 卡特尔。
B 辛迪加
C 托拉斯
D 康采恩
解析：解析：（1）卡特尔：通过生产或经营同类产品的一些大型垄断企业之间较稳定的垄断协议形成的、在生产或流通的某个或某些环节实行垄断的组织形式。（2）辛迪加：把一些在法律上和生产上还是独立的大垄断企业在流通环节上统一起来，进行垄断经营的组织形式。（3）托拉斯：垄断了某种商品生产和经营全过程的独立的大型垄断企业。（4）康采恩：由一两个特大型垄断企业为核心，联合了各方面各领域一大批企业所形成的超大型垄断集团。

国家垄断资本主义是指（  ）（A）
A 资本主义国家与垄断资本相结合的资本主义
B 垄断的工业资本与垄断的银行资本相结合的资本主义
C 资本主义国家的中央银行与商业银行相结合的资本主义
D 资本主义国家的国有经济与非国有经济相结合的资本主义
解析：解析：国家垄断资本主义是指资本主义国家与垄断资本相结合的资本主义。

马克思对资本主义的历史过渡性作了精辟而辩证的论述，他所说的“资本主义的历史任务和存在理由”是指（  ）（D）

下列选项中属于共产主义社会显著特征的是（  ）（D）
A 按生产要素分配
B 各尽所能，按劳分配

马克思主义认为，共产主义社会将经历两个不同的发展阶段，第一个阶段是（  ）（C）
A 产品经济社会
B 商品经济社会
C 社会主义社会
D 自然经济社会
解析：解析：共产主义社会将经历两个不同的发展阶段：第一阶段或低级阶级是社会主义社会、第二阶段或高级阶段是共产主义社会。
