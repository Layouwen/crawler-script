2015年10月马克思主义基本原理概论真题（选择题）

<p>共产主义社会的分配原则是</p>（B）
A 按劳分配
B 按需分配
C 平均分配
D 定量分配
解析：<p>共产主义社会实行按需分配原则，社会主义实行按劳分配原则。</p>

<p>下列选项中体现社会主义的本质属性和要求的是</p>（A）
A 促进人的全面发展和社会和谐
B 实行高度发达的商品经济
C 建立完全纯粹的公有制经济
D 实行严格管控的计划经济
解析：<p>邓小平把社会主义的本质概括为“解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕”。中国共产党人在进一步的探索中认识到，推动科学发展、促进社会和谐、实现人的全面发展，是社会主义的本质属性和要求。</p>

<p>无产阶级革命的发生归根结底是</p>（A）
A 生产力的发展水平决定的
B 生产关系的性质决定的
C 被统治阶级的意愿决定的
D 统治阶级的压迫决定的
解析：<p>从19世纪马克思和恩格斯所处时代的工人运动来看，革命的发生归根结底是由生产力发展水平决定的。</p>

<p>标志着科学社会主义理论公开面世的纲领性文件是</p>（B）
A 《共产主义原理》
B 《共产党宣言》
C 《哥达纲领批判》
D 《国家与革命》
解析：<p>1847年底马克思和恩格斯受世界上第一个无产阶级政党“共产主义者同盟”的委托起草并于1848年2月发表的《共产党宣言》，是社会主义思想史上的第一个纲领性文件，它标志着科学社会主义理论公开面世。</p>

<p>垄断资本主义的历史作用是</p>（A）
A 为一个更高级的生产形式创造物质条件
B 建立完善的市场体系和市场机制
C 实现生产社会化和经济全球化
D 不断发展社会的民主与法制
解析：<p>马克思指出：“发展社会劳动生产力，是资本主义的历史任务和存在理由。资本正是以此不自觉地为一个更高级的生产形式创造物质条件。”这是对资本主义历史过渡性最精辟而辩证的论述。</p>

<p>资本主义国家宏观经济管理与调节的总任务是</p>（C）
A 实现充分就业
B 稳定币值、汇率和物价
C 促进市场总供需平衡
D 推动经济发展与增长
解析：<p>国家垄断资本主义条件下的宏观经济调控主要有： （1）国家干预经济的必要性。 （2）资本主义国家宏观经济管理与调节的任务、目标。资本主义国家宏观经济管理与调节的总任务，是促进市场总供给和总需求的平衡。 （3）资本主义国家对经济的调控方式与手段。国家垄断资本主义宏观经济调控的主要政策。</p>

<p>垄断资本主义国家的金融资本是指</p>（D）
A 资本主义国家中处于垄断地位的银行资本
B 资本主义国家金融领域中流通的所有资本的总和
C 资本主义国家银行和非银行金融机构资本的总和
D 垄断的工业资本和银行资本融合形成的资本
解析：<p>金融资本是垄断资本主义国家中社会经济生活各个方面实际上的统治者，它是由垄断的工业资本和银行资本，通过金融联系、资本参与和人事参与，融合或混合生长构成的一种新的资本形式。</p>

<p>垄断资本主义社会大体可分为</p>（B）
A 一般垄断资本主义和私人垄断资本主义两个阶段
B 私人垄断资本主义和国家垄断资本主义两个阶段
C 一般垄断资本主义、私人垄断资本主义和国家垄断资本主义三个阶段
D 私人垄断资本主义、国家垄断资本主义和国际垄断资本主义三个阶段
解析：<p>垄断资本主义阶段以第二次世界大战结束为界，划分为私人垄断与国家垄断两个发展阶段，国家垄断资本主义是垄断资本主义发展的一个新阶段。</p>

<p>资本主义政治制度的核心是</p>（C）
A 政党制度
B 法律制度
C 国家制度
D 选举制度
解析：<p>资本主义政治制度就是资产阶级为实现其阶级专政而采取的统治方式和方法及各相关制度的总和。它主要包括资本主义的国家制度、政党制度、选举制度、三权分立制度、民主制度等等，其中国家制度是资本主义政治制度的核心。</p>

<p>社会总产品是社会各个物质生产部门在一定时期(通常是一年)内生产的</p>（C）
A 全部剩余价值的总和
B 全部生产资料的总和
C 全部物质资料的总和
D 全部消费资料的总和
解析：<p>本题考查的是社会总产品的含义，需要考生识记。社会总产品就是社会各个物质生产部门在一定时期（通常是以年为单位）内所生产的全部物质资料的总和。</p>

<p>资本积累的源泉是</p>（B）
A 使用价值
B 剩余价值
C 价值
D 交换价值
解析：<p>剩余价值是资本积累的源泉。在资本主义制度下，资本积累具有客观必然性。</p>

<p>以私有制为基础的商品经济的基本矛盾是</p>（D）
A 使用价值和价值的矛盾
B 具体劳动和抽象劳动的矛盾
C 价值和交换价值的矛盾
D 私人劳动和社会劳动的矛盾
解析：<p>以生产资料私有制为基础的商品经济的基本矛盾，是私人劳动和社会劳动的矛盾。</p>

<p>人的本质在其现实性上是</p>（D）
A 人的道德品性
B 人的经济地位
C 人的价值追求
D 一切社会关系的总和
解析：<p>人的具体本质在其现实性上是一切社会关系的总和，它是由社会关系决定的。</p>

<p>阶级作为一种社会现象首先是一个</p>（A）
A 经济范畴
B 政治范畴
C 文化范畴
D 交往范畴
解析：<p>阶级作为一种社会现象，首先是一个经济范畴，是一些经济集团，划分阶级的唯一标准是经济标准，不能用政治标准和思想标准划分阶级。</p>

<p>在生产关系的诸要素中起决定作用的是</p>（D）
A 产品的分配方式
B 人们在生产中的地位
C 产品的交换方式
D 生产资料的所有制形式
解析：<p>生产关系是指人们在物质生产过程中结成的经济关系。它包括三项内容： （1）生产资料的所有制形式； （2）人们在生产中的地位及其相互关系（包括交换）； （3）产品的分配方式。 其中生产资料所有制形式起着决定作用，它是整个生产关系的基础。</p>

<p>“作为观念形态的文艺作品，都是一定的社会生活在人类头脑中的反映的产物”，这句话强调的是</p>（A）
A 社会意识根源于社会存在
B 社会意识具有历史继承性
C 社会存在根源于社会意识
D 社会意识具有相对独立性
解析：<p>人们的社会意识既不是从天上掉下来的，也不是头脑里主观自生的，而是在实践的基础上对社会存在的反映。有什么样的社会存在，就会产生什么样的社会意识。因此，根据题干可以判断出，社会意识根源于社会存在。</p>

<p>“哲学家们只是用不同的方式解释世界，问题在于改变世界”的提出者是</p>（C）
A 黑格尔
B 费尔巴哈
C 马克思
D 恩格斯
解析：<p>本题是对人名的考查，需要考生熟记。本题中这句话的提出者是马克思。</p>

<p>在真理标准问题上坚持辩证法，就是要承认实践标准</p>（D）
A 既是抽象的又是具体的
B 既是必然的又是偶然的
C 既是普遍的又是特殊的
D 既是绝对的又是相对的
解析：<p>真理是客观的，又是绝对的和相对的。承认真理的客观性，这是在真理问题上坚持了唯物论；承认真理既是绝对的又是相对的，这是在真理问题上坚持了辩证法。任何真理都是客观性、绝对性和相对性的统一，这就是真理观上唯物论与辩证法的统一。</p>

<p>“感觉只解决现象问题，理论才解决本质问题”，这句话说明</p>（B）
A 感性认识是微不足道的
B 感性认识有待于发展到理性认识
C 理性认识是唯一可靠的
D 理性认识有待于深化为感性认识
解析：<p>在认识过程中，感性认识非常重要，但我们不能停留于、满足于感性认识，有待于发展到理性认识。因为“感觉到了的东西，我们不能立刻理解它，只有理解了的东西才更深刻地感觉它。感觉只解决现象问题，理论才解决本质问题。”</p>

<p>能动反映论区别于机械反映论的根本标志是</p>（D）
A 肯定思维能正确地反映
存在
B 认为认识过程是纯粹的观念
创造
C 肯定思维对存在的决定作用
D 认为认识过程是反映和创造的统一
解析：<p>认识的过程是反映与创造的统一，是辩证唯物论的反映论的能动性的突出表现，也是区别于机械唯物论反映论的根本标志。</p>

<p>意识对于人的生理活动具有一定影响作用，这表明</p>（A）
A 意识对物质具有能动性
B 意识是客观事物本身
C 意识对物质具有决定性
D 意识是人脑的机能
解析：<p>意识的能动性主要表现在以下四个方面： （1）意识活动具有目的性和计划性。 （2）意识活动具有主动性、创造性。 （3）意识对于人的生理活动具有一定影响作用。 （4）意识能通过指导实践改造客观物质世界。</p>

<p>唯物辩证法的总特征有两个，一个是联系的观点，另一个是</p>（A）
A 发展的观点
B 对立统一的观点
C 实践的观点
D 辩证否定的观点
解析：<p>唯物辩证法认为，物质世界是普遍联系和永恒发展的。联系的观点和发展的观点是唯物辩证法的总特征。</p>

<p>马克思主义认为社会生活在本质上是实践的，这是因为</p>（D）
A 实践主体处于一定的社会关系中
B 实践具有客观性、能动性和社会历史性
C 实践是个人日常生活的活动
D 实践是人类社会产生、存在和发展的基础
解析：<p>马克思主义认为，实践是人类社会产生、存在和发展的基础，是社会生活的本质。</p>

<p>划分唯物主义和唯心主义的唯一标准是，是否承认</p>（C）
A 世界的统一性和多样性
B 联系的客观性和普遍性
C 物质第一性意识第二性
D 思维和存在具有同一陛
解析：<p>哲学的基本问题包括两个方面的内容。第一方面是物质和意识哪个是本原、哪个是第一性的问题。如何回答这个问题是划分唯物主义和唯心主义的标准。凡是认为物质是本原的、第一性的，意识是派生的、第二性的哲学，属于唯物主义；凡是认为意识是本原的第一性的，物质是派生的、第二性的哲学，属于唯心主义。第二方面是思维和存在的同一性问题，这在哲学上属于认识论问题。</p>

<p>马克思主义哲学以前欧洲哲学的全部积极内容，都是马克思主义哲学的理论来源，其中作为马克思主义哲学直接理论来源的是</p>（B）
A 古希腊罗马哲学
B 德国古典哲学
C 近代英国哲学
D 近代法国哲学
解析：<p>马克思主义哲学的直接理论来源主要有德国古典哲学、英国古典政治经济学和英法两国的空想社会主义学说。</p>
