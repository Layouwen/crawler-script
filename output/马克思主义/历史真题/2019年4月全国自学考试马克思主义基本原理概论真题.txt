2019年4月全国自学考试马克思主义基本原理概论真题（选择题）

<p>资本主义经济危机从实质上说就是</p>（B）
A 生产绝对过剩的危机
B 生产相对过剩的危机
C 生产不足的危机
D 买卖脱节的危机
解析：<p>资本主义经济危机从实质上说就是生产相对过剩的危机。生产相对过剩是指相对于劳动人民有支付能力的需求来说社会生产的商品显得过剩，而不是与劳动人民的实际需求相比的绝对过剩。选B。</p>

<p>资本主义政治制度的核心是</p>（C）
A 政党制度
B 选举制度
C 国家制度
D <p>文官制度</p>
解析：<p>资本主义政治制度主要包括资本主义的国家制度、政党制度、选举制度、三权分立制度、民主制度等等，其中国家制度是资本主义政治制度的核心。</p>

<p>当代国际垄断组织的最主要形式是</p>（D）
A 国际卡特尔
B 国际辛迪加
C 国际托拉斯
D 跨国公司
解析：<p>当代国际垄断组织的形式以跨国公司和国家垄断资本主义的国际垄断同盟为主。</p>

<p>垄断资本主义可分为</p>（A）
A 私人垄断和国家垄断两阶段
B 私人垄断、国家垄断和国际垄断三阶段
C 私人垄断和一般垄断两阶段
D 私人垄断、一般垄断和国家垄断三阶段
解析：<p>把垄断资本主义阶段以第二次世界大战结束为界，划分为私人垄断与国家垄断两个发展阶段。</p>

<p>垄断利润的真正来源是</p>（D）
A 垄断行为所创造的更多市场价值
B 垄断企业在自由市场上的贱买贵卖
C 资本主义国家对国民收入的再分配
D 工人阶级和劳动人民所创造的剩余价值
解析：<p>垄断本身不能创造价值和剩余价值，垄断资本所获得的高额利润，归根到底来自工人阶级和劳动人民所创造的剩余价值。</p>

<p>资本主义国家对国民经济的管理和调节，主要依靠</p>（A）
A 经济手段
B 法律手段
C 行政手段
D 计划手段
解析：<p>资本主义国家对经济的调控 方式，一个是国家对国民经济的直接调节、控制和介入，如对国有企业和国有私人合营企业的经营、管理和调控，这在整个国家垄断资本主义的经济调控中所占比重不大。另一个是国家对国民经济的间接调节与控制，主要是运用经济手段，通过经济政策的制定、经济杠杆的运用，还可以通过计划的指导，间接地引导私人资本按照国家垄断资本主义所希望的方向发展</p>

<p>世界上第一个无产阶级政党是</p>（C）
A 中国共产党
B 德国社会民主工党
C 共产主义者同盟
D 俄国社会民主工党
解析：<p>马克思、恩格斯于1847年创建的共产主义者同盟,是世界上第一个无产阶级政党。它是各国工人政党中最先进和最坚决的部分。</p>

<p>在2018年3月的全国人民代表大会上，作为党的指导思想被首次载入宪法的是</p>（D）
A 毛泽东思想
B 邓小平理论
C “三个代表”重要思想
D 习近平新时代中国特色社会主义思想
解析：<p>2018年3月的全国人民代表大会上，习近平新时代中国特色社会主义思想载入了宪法，宪法修正案明确了习近平新时代中国特色社会主义思想在国家政治和社会生活中的指导地位。</p>

<p>实现共产主义社会的根本条件和基础是</p>（A）
A 社会生产力的高度发展
B 人与自然关系的高度和谐
C 人们道德水平的极大提升
D 人们法律意识的极大提高
解析：<p>随着社会生产力的发展，社会主义的生产关系和上层建筑不断调整和完善，推动着社会主义社会逐步向前发展。依据人类社会的发展规律，社会主义社会也是在生产力和生产关系的矛盾运动中不断向前发展的。</p>

<p>当前我国人民的历史使命和共同理想是</p>（A）
A 建设中国特色社会主义
B 实现共产主义
C 实现全人类的彻底解放
D 实现人类大同
解析：<p>我国当前正处于社会主义初级阶段，建设中国特色社会主义是我国人民的历史使命和共同理想。</p>

<p>决定资本主义社会劳动力价值的因素中，对各因素都有影响和渗透作用的是</p>（C）
A 维持生存的生活资料的价值
B 延续和养育后代所需的费用
C 历史和道德的因素
D 教育和培训的费用
解析：<p>劳动力的价值决定有一个重要的特点，就是它包括历史和道德的因素。这就是说，雇佣劳动者的必要生活资料的种类和数量，要受一定历史条件下的经济和文化发展水平以及各个国家风俗和习惯的制约。</p>

<p>具体劳动</p>（C）
A 反映社会生产关系
B 是劳动的社会属性
C 反映人与自然的关系
D 形成商品的价值
解析：<p>具体劳动是生产不同使用价值的不同性质和不同形式的劳动。由生产的目的、操作方式、劳动对象、手段和结果决定。从这一角度看劳动，它是具体的，故称具体劳动。在商品生产条件下，具体劳动是生产商品的劳动二重性的一个方面。任何社会，人类为了自身的生存和发展，都必须从事各种具体劳动，改造各种特殊的自然物质，生产出各种使用价值，满足生活和生产中的各种需要。具体劳动反映人与自然的关系。选C。</p>

<p>唯物史观认为，区分普通个人和历史人物主要是看他们</p>（B）
A 物质财富的多寡
B 历史作用的大小
C 天赋才能的有无
D 社会阅历的深浅
解析：<p>个人按其对历史发展作用的大小，可以区分为普通个人和历史人物。</p>

<p>推动社会历史发展的动力是多方面的，其中最基本的动力是</p>（D）
A 科学和技术
B 广大人民群众
C 革命和改革
D 社会基本矛盾
解析：<p>社会基本矛盾是社会发展的基本动力。</p>

<p>在现代生产力系统中，分工协作和经济管理属于</p>（B）
A 实体性因素
B 运筹性因素
C 渗透性因素
D 准备性因素
解析：<p>运筹性的综合因素，包括分工协作、经济管理、预测决策等。</p>

<p>下列各项正确说明人口因素在社会发展中作用的是</p>（D）
A 人口因素决定社会制度的性质
B 人口因素为人类提供生存的场所
C 人口因素决定社会制度的更替
D 人口状况能加速或延缓社会发展
解析：<p>人口因素是社会物质生活的必要条件之一，对社会发展起着制约和影响作用，主要表现在：第一，一定数量的人口是社会物质生产的必要前提。第二，人口状况能加速或延缓社会的发展。</p>

<p>真理的发展是一个</p>（B）
A 由主观真理走向客观真理的过程
B 由相对真理走向绝对真理的过程
C 由具体真理走向抽象真理的过程
D 由特殊真理走向终极真理的过程
解析：<p>真理的发展过程就是由相对真理走向绝对真理的过程。</p>

<p>下列各项不属于认识客体的是</p>（D）
A 已经探测到的引力波
B 已经揭秘的史前文明
C 已经观测到的银河系
D 尚未发现的地下矿藏
解析：<p>构成认识客体的主要有自然客体、社会客体和精神客体。认识客体——人的认识活动所指向的对象。</p>

<p>马克思主义认识论的首要的和基本的观点是</p>（A）
A 实践的观点
B 发展的观点
C 历史的观点
D 辩证的观点
解析：<p>实践的观点是马克思主义哲学认识论首要的和基本的观点。</p>

<p>“揠苗助长”的寓言说明的哲学道理是</p>（D）
A 意识的能动作用都是积极的
B 发挥主观能动性可以改变规律
C 意识的能动作用总是巨大的
D 发挥主观能动性不能违背规律
解析：<p>尊重客观规律是正确发挥主观能动性的前提。如果违背客观规律，盲目蛮干，必然受到客观规律的无情惩罚。违背了规律，越是发挥主观能动性，遭受的挫折和失败就越严重。“揠苗助长”的寓言就说明了这个道理。</p>

<p>建设中国特色社会主义既要坚持科学社会主义一般原则，又要结合中国实际。这符合</p>（A）
A 矛盾的普遍性和特殊性辩证关系的原理
B 矛盾的同一性和斗争性辩证关系的原理
C 社会历史发展的决定性和选择性统一的原理
D 社会历史发展的前进性和曲折性统一的原理
解析：<p>建设中国特色社会主义既要坚持科学社会主义一般原则,又要结合中国实际。这符合矛盾的普遍性和特殊性辩证关系的原理。</p>

<p>世界观是人们对</p>（B）
A 自然界的根本看法和根本观点
B 整个世界的根本看法和根本观点
C 人类社会的根本看法和根本观点
D 精神世界的根本看法和根本观点
解析：<p>世界观，就是人们对包括自然、社会和人类思维在内的整个世界的根本看法和根本。</p>

<p>客观实在性作为物质的唯一特性，表明它是</p>（B）
A 具体物质的个性
B —切物质的共性
C 自然物质的个性
D 自然物质的共性

<p>与时倶进是马克思主义的</p>（C）
A 理论前提
B 理论核心
C 理论品质
D 理论特征

<p>马克思主义哲学的直接理论来源是</p>（C）
