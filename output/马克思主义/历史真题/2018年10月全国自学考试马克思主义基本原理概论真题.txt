2018年10月全国自学考试马克思主义基本原理概论真题（选择题）

<p>斯密和李嘉图是</p>（B）
A 德国古典哲学的代表人物
B 英国古典政治经济学的代表人物
C 英国空想社会主义的代表人物
D 法国空想社会主义的代表人物
解析：<p>英国古典政治经济学的代表人物是斯密和李嘉图。A选项德国古典哲学的代表人物康德、费希特、谢林、黑格尔和费尔巴哈。C选项英国空想社会主义的代表人物是欧文。D选项法国空想社会主义的代表人物是圣西门和傅里叶。</p>

<p>关于马克思主义的产生，下列说法错误的是</p>（D）
A 马克思主义是历史时代的产物
B 马克思主义是实践经验的总结
C 马克思主义是科学成果的升华
D 马克思主义是先天智慧的结晶
解析：<p>马克思主义是集体智慧的结晶。</p>

<p>哲学基本问题第二方面内容主要回答的是</p>（B）
A 世界的本质是什么的问题
B 世界可否被认识的问题
C 世界的存在是怎样的问题
D 世界有否统一性的问题
解析：<p>哲学基本问题是思维和存在的关系问题，或者说是精神和物质、意识和物质的关系问题。哲学基本问题第一方面是精神和物质何者为第一性、何者是本原的问题。对这一问题的不同回答，是划分唯物主义和唯心主义两大哲学基本派别的唯一标准。第二方面，精神和物质是否具有同一性，即人的意识能否正确认识现实世界的问题。对这一问题的不同回答，是划分可知论和不可知论的标准。可知论认为人们能够正确认识现实世界；不可知论认为人们不可能认识现实世界。</p>

<p>联系的观点和发展的观点是唯物辩证法的</p>（A）
A 总特征
B 精髓
C 实质
D 核心
解析：<p>唯物辩证法的总特征是联系和发展。</p>

<p>“一把钥匙开一把锁”。这句话强调的是，在解决复杂问题时</p>（C）
A 要具体地分析矛盾的斗争性
B 要具体地分析矛盾的同一性
C 要具体地分析矛盾的特殊性
D 要具体地分析矛盾的普遍性
解析：<p>矛盾的特殊性,构成了一事物区别于他事物的特殊本质,是世界上的事物之所以千差万别的内在原因或根据，故选项C符合题意。</p>

<p>规律的两个特点是</p>（D）
A 普遍性和特殊性
B 客观性和自发性
C 盲目性和自觉性
D 客观性和普遍性
解析：<p>规律具有两个特点 1.客观性：规律是运动着的事物本身固有，不以任何人的意志为转移。 2.普遍性：不是只在个别、特殊的事物和现象起作用，而是在较大范围和领域起作用。</p>

<p>下列选项中，正确揭示了认识的本质的是</p>（A）
A 认识是主体对客体的能动反映
B 认识是主体的主观创造
C 认识是主体对客体的先验反映
D 认识是主体的内心体验
解析：<p>认识的本质的是主体对客体的能动反映</p>

<p>人们认识世界的目的在于</p>（B）
A 满足猎奇心理
B 改造客观世界
C 实现自我价值
D 创造客观规律
解析：<p>人类认识世界的最终目的是为了改造世界，而改造世界的活动又包括两个方面：改造客观世界和改造主观世界。</p>

<p>在真理的检验标准问题上，下列各项中说法正确的是</p>（D）
A 多数人认可的就是真理
B 个人觉得有用的就是真理
C 少数人坚持的就是真理
D 在实践中证实的才是真理
解析：<p>检验真理的标准只能是社会实践，理论与实践的统一是马克思主义的一个最基本的原则，任何理论都要不断接受实践的检验。这是从根本理论上对“两个凡是”的否定。</p>

<p>社会意识具有复杂而精微的结构。从反映社会存在的程度和特点来看，社会意识包括</p>（A）
A 社会心理和思想体系
B 意识形态和非意识形态
C 个体意识和群体意识
D 主流意识和非主流意识
解析：<p>社会意识具有复杂而精微的结构，由诸多层次和因素构成。从反映社会存在的程度和特点来看，社会意识包括社会心理和思想体系两个层次；从社会意识主体的范围来看，它可以分为个体意识和群体意识。</p>

<p>划分阶级所依据的根本标准是各社会集团的</p>（B）
A 政治面貌
B 经济地位
C 宗教信仰
D 价值取向
解析：<p>通过本题要把握阶级的划分根源在于经济地位不同，这是划分阶级的唯一标准，不能用政治或者思想标准划分。所以阶级之间的斗争是物质利益的对立斗争。</p>

<p>改革不仅存在于社会主义社会中，而且存在于其他社会形态中。这说明改革具有</p>（C）
A 主体能动性
B 历史局限性
C 普遍性
D 特殊性
解析：<p>社会改革的普遍性是指社会改革不仅仅存在于社会主义社会中，而且存在于有史以来的各种社会制度中。都在一定程度上推动了当时社会经济文化的繁荣与发展。</p>

<p>“社会形态的发展是一个自然历史过程”。这句话说的是</p>（D）
A 社会规律和自然规律没有差别
B 社会发展排斥主体选择的作用
C 社会发展取决于主体自由选择
D 社会发展具有不依人的意志为转移的客观规律性
解析：<p>人类社会是一个自然历史过程，这是唯物史观的基本观点。人类社会虽然有人的自觉的、能动的作用，但是，在根本上，人类社会同自然界是一致的，都是客观的、合乎规律的辩证发展过程。（1）人类社会和自然界一样，都不是人的意志的自由创造，而是物质的表现形态，是不以于人们的意志为转移的客观存在；（2）人类社会和自然界一样，不是凝固不变的，而是处于经常发展变化的矛盾运动过程中的；（3）人类社会和自然界一样，都有自身内在的由低级向高级发展的客观规律性。故选项D符合题意。</p>

<p>简单商品经济的基本矛盾是指</p>（C）
A 价值和使用价值的矛盾
B 价值和交换价值的矛盾
C 私人劳动和社会劳动的矛盾
D 具体劳动和抽象劳动的矛詹
解析：<p>生产商品的劳动具有私人劳动和社会劳动二重性，私人劳动与社会劳动的矛盾也成为简单商品经济的基本矛盾。这是因为： (1)私人劳动与社会劳动的矛盾决定了商品经济的本质和它的发展过程。 (2)私人劳动与社会劳动的矛盾决定和影响着私有制商品经济的其他一切矛盾。 (3)私人劳动与社会劳动的矛盾对于商品生产者至关重要。</p>

<p>资本有机构成的提高一般是以</p>（B）
A 社会总资本的增大为前提
B <p>生产资料的增大为前提</p>
C 流通资本的增大为前提
D 单个资本的增大为前提
解析：<p>由资本技术构成所决定,并且反映资本技术构成变化的资本价值构成叫做资本有机构成,可用公式c:v来表示。在资本积累过程中,由于追求更多剩余价值的内在动力和迫于竞争的外在压力,资本家必然会不断改进企业的生产技术装备,提高劳动生产率,结果在全部资本中,不变资本所占比重增大,可变资本所占比重缩小,从而导致资本有机构成的提高。所以,资本有机构成的不断提高,是资本主义经济发展的必然趋势。(注意,此题有争议。参考答案选B,单从有机构成公式来看,选B没错。但究其根本,是资本家的资本积累了,能够投入更多资金购买先进设备,提高技术水平,从这个角度来说应该选D。</p>

<p>从资本周转的角度来考察资本的构成，划分为固定资本和流动资本的是</p>（A）
A 生产资本
B 货币资本
C 商品资本
D 流通资本
解析：<p>生产资本从资本周转的角度划分为固定资本和流动资本。</p>

<p>资本主义国家的民主制度存在形式上平等而事实上不平等的矛盾，根本原因在于</p>（D）
A 宪法是由资产阶级制定的
B 民主的原则是与实践相脱节的
C 劳动人民不能参与国家政权的管理
D 民主权利的实现受到资本主义生产资料私有制的限制
解析：<p>政治根源于经济。资产阶级在制定宪法时，之所以要用“全民性”来掩盖其民主的阶级性，使资本主义民主制度中存在形式上平等和事实上不平等的矛盾，其根源在于生产资料资本主义私有制。可见，D项是经济方面的原因，是根本原因，A项是造成上述矛盾的政治方面的原因和直接原因。题项B和C是题干内容的变相重复，或者说都是题干内容的表现和注释，而不是原因。</p>

<p>垄断利润的水平一般会</p>（B）
A 低于平均利润
B 高于平均利润
C 等于平均利润
D 等于成本价格
解析：<p>垄断利润的水平一般高于平均利润。垄断资本的实质在于获取垄断利润，垄断利润是垄断资本家凭借其在社会生产和流通中的垄断地位而获得的超过平均利润的高额利润。垄断资本所获得的高额利润，归根到底来自无产阶级和其他劳动人民创造的剩余价值。</p>

<p>在经济危机和萧条阶段，资本主义国家向私人企业注入一部分国有资金以维持其生存的经济行为属于</p>（B）
A 建立资本主义国有经济和国有企业的一种方式
B 国家和私人资本在企业内部结合的一种方式
C 国家和私人资本在企业外部结合的一种方式
D 资本主义国家实行经济计划的一种方式
解析：<p>本题考查国家垄断资本主义的主要形式。国家与私人垄断资本在企业内部的结合，即形成国家与私人共有的垄断资本。</p>

<p>资本主义国家对国民经济的间接调控主要是运用</p>（A）
A 经济手段
B 法律手段
C 行政手段
D 计划手段
解析：<p>资本主义国家对经济的调控手段，主要是经济手段和法律手段，国家垄断资本主义宏观经济管理和调控目标的实现，主要是通过国家调节市场、市场引导企业这样一种经济调节机制实现的，这也是国家对经济的管理调控以经济手段为主的具体体现。</p>

<p>垄断资本主义的历史作用是</p>（A）
A 为一个更高级的生产形式创造物质条件
B 建立完善的市场体系和市场机制
C 实现生产社会化和经济全球化
D 不断发展社会的民主与法制
解析：<p>马克思指出：“发展社会劳动生产力，是资本主义的历史任务和存在理由。资本正是以此不自觉地为一个更高级的生产形式创造物质条件。”这是对资本主义历史过渡性最精辟而辩证的论述。</p>

<p>在社会主义思想史上，标志着科学社会主义理论公开面世的第一个纲领性文件是</p>（A）
A 《共产党宣言》
B 《法兰西内战》
C 《社会主义从空想到科学的发展》
D 《哥达纲领批判》
解析：<p>1848年2月发表的《共产党宣言》，是社会主义思想史上的第一个纲领性文件，它标志着科学社会主义理论公开面世，从而使社会主义由空想变为科学。</p>

<p>根据邓小平的概括，下列选项不属于社会主义本质的是</p>（C）
A 解放生产力，发展生产力
B 消灭剥削，消除两极分化
C 允许个别人先富起来
D 最终达到共同富裕
解析：<p>关于社会主义的本质，邓小平同志有一个经典的概括，这就是人们熟知的“社会主义的本质，是解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕”。这一概括，深刻地阐明了社会主义的本质，贯穿于邓小平理论的各个方面，蕴含着十分丰富的思想，需要我们全面准确地加以理解。</p>

<p>作为人类社会发展的最高社会形态，共产主义社会的物质基础是</p>（D）
A 与任何繁荣的封建社会相当的社会生产力
B 与一般资本主义社会相当的社会生产力
C 与发达资本主义国家相当的社会生产力
D 远远高于以往一切社会的高度发达的社会生产力
解析：<p>马克思、恩格斯认为，共产主义社会形态，按其成熟程度，可以分为低级和高级两个阶段，即社会主义社会和共产主义社会。根据马克思主义经典作家的论述，共产主义社会具有以下基本特征：物质财富极大丰富，消费资料按需分配。在共产主义社会里，生产力高度发展，社会物质财富不断涌流，社会产品极大丰富，达到了可以满足整个社会及其成员需要的程度。故选项D符合题意。</p>

<p>我们实现人生价值的最终归宿是</p>（B）
A 社会主义共同理想
B 共产主义远大理想
C 天下大同的社会理想
D 克己奉公的道德理想
解析：<p>共产主义理想是为基本内容的奋斗目标，是共产党人的最高理想。它是由通过批判地吸收空想社会主义的思想成果，运用辩证唯物主义和历史唯物主义的科学世界观分析社会发展的客观规律，认真总结工人运动的实践经验而提出来的一种光辉的社会理想。党的十六大章程指出:&quot;党的最高理想和最终目标是。&quot;我们党之所以以实现共产主义为最终奋斗目标，是因为共产主义是人类社会最和谐、最美好、最高级的社会形态。</p>
