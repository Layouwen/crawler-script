2017年4月全国自学考试马克思主义基本原理概论真题（选择题）

<p>当代中国人民的历史使命是</p>（A）
A 建设中国特色社会主义
B 实现社会绝对的公平正义
C 实现国家完全的文明和谐
D 实现公民彻底的自由平等
解析：<p>我国当前正处于社会主义初级阶段，建设中国特色社会主义是我国人民的历史使命和共同理想。</p>

<p>劳动不再仅仅是一种谋生手段而成为人们生活第一需要的社会是</p>（B）
A 社会主义社会
B <p>
	共产主义社会
</p>
C 资本主义社会
D 信息网络社会
解析：<p>第七章第一节马克思：“在共产主义高级阶段......在劳动不再仅仅是一种谋生手段而成为人们生活第一需要的社会......”</p>

<p>马克思主义政党的产生有两个条件, 一是工人运动的发展,二是</p>（B）
A 社会贫富分化严重
B 科学社会主义理论的传播
C 社会矛盾严重激化
D 无产阶级领袖人物的出现24
解析：<p>马克思主义政党的产生有两个条件, 一是工人运动的发展,二是科学社会主义理论的传播。</p>

<p>社会主义社会的改革是</p>（D）
A 社会主义基本经济制度的改变
B 社会主义核心价值观的根本改变
C 社会主义基本政治制度的改变
D 社会主义制度的自我完善和发展
解析：<p>基本经济制度、核心价值观还有基本政治制度都没有变。</p>

<p>资本主义经济的发展为向更高级的生产方式过渡创造物质条件, 这主要表现在</p>（C）
A 无产阶级队伍的壮大
B 科学技术水平的不断提高
C 社会化大生产的发展
D 市场经济体制的发展完善
解析：<p>资本主义条件下生产社会化的全面发展，推动了资本主义关系的日益社会化，为向社会主义转变准备了日益完备的社会经济基础。</p>

<p>下列选项中不属于当代资本主义生产力方面新变化的是</p>（D）
A 生产者中脑力劳动者的比重增加
B 机器生产由“三机系统''发展为“四机系统”
C 生产管理中增加了计算机信息技术等新手段和工具
D 在生产资料所有制上出现了法人所有制等新形式
解析：<p>从生产力方面看，战后发达资本主义国家，在生产力方面取得了长足的发展。在产业结构上，出现了转向信息化、服务化和高科技化的趋势；在生产要素的变化上，就劳动者来说，当代发达资本主义国家中劳动力结构变化很大，脑力劳动者比例攀升，素质显著提高；劳动工具也发生了革命性变革，由传统机器生产的动力机、传动机、工作机组成的“三机系统”，发展为增加了电脑控制机的“四机系统”，生产工具的智能化趋势日渐加强；就劳动对象来说，新变化也层出不穷：人工合成材料的面世，新资源的开发利用，尤其是信息资源这一非物质资源的广泛应用，从根本上改变了劳动对象的物质范围；此外，作为生产力重要组成部分的生产管理，由于电子计算机、信息技术、控制论、系统工程等的运用，而获得了新的手段和工具。D是属于生产力关系方面的变化。</p>

<p>垄断利润的真正来源是</p>（C）
A 垄断行为所创造的更多市场价值
B 垄断势力在自由市场上的巧取豪夺
C 工人阶级和劳动人民所创造的剩余价值
D 生产和非生产领域劳动者共同创造的物质财富
解析：<p>垄断利润的真正来源于工人阶级和劳动人民所创造的剩余价值。来源大体有几个方面：第一，来自对本国无产阶级和其他劳动人民的剥削；第二，由于垄断资本可以通过垄断高价和垄断低价来控制市场，使得它能获得一些其他企业，特别是非垄断企业的利润；第三，通过加强对其他国家劳动人民的剥削和掠夺，获得海外利润；第四，通过资本主义国家政权进行有利于垄断资本的再分配，从而将劳动人民创造的国民收入的一部分变成垄断资本的收入。</p>

<p>在国家垄断资本主义阶段之前,资本主义处于</p>（B）
A 自由竞争资本主义阶段
B 私人垄断资本主义阶段
C 垄断竞争资本主义阶段
D 金融垄断资本主义阶段
解析：<p>在国家垄断资本主义阶段之前,资本主义处于私人垄断资本主义阶段。</p>

<p>资本主义国家的政体主要有</p>（B）
A 两党制和多党制两种
B 君主立宪制和民主共和制两种
C 一党制、两党制和多党制三种
D 一党制、两党制和多党制三种
解析：<p>资本主义国家的政体是指资本主义国家的政权机构的构成形式，主要有君主立宪制和民主共和制两种形式。</p>

<p>生产资料和劳动力实物构成上的比例是资本的</p>（C）
A 有机构成
B 价值构成
C 技术构成
D 数量构成
解析：<p>
	资本的构成可以从两方面来考察。一方面，从物质形态看，资本是由一定数量的生产资料和劳动力所构成，它们之间的比例是由生产技术水平所决定的。这种反应生产技术水平的生产资料和劳动力之间的比例，叫做资本技术构成。另一方面，从价值形态看，资本是由一定数量的不变资本和可变资本构成的，它们之间的比例叫做资本价值构成。资本基础构成决定资本价值构成。这种由资本技术构成变化的资本价值构成，叫做资本有机构成。</p>

<p>价值规律作用的表现形式是</p>（A）
A 商品价格围绕价值波动
B 商品价格经常低于价值
C 价值围绕商品价格波动
D 商品价格往往高于价值
解析：<p>价值规律作用的表现形式是商品价格围绕价值波动。</p>

<p>简单商品经济的基本矛盾是</p>（D）
A 使用价值和价值的矛盾
B 价值和交換价值的矛盾
C 具体劳动和抽象劳动的矛盾
D 私人劳动和社会劳动的矛盾
解析：<p>简单商品经济的基本矛盾是私人劳动和社会劳动的矛盾。</p>

<p>区分普通个人和历史人物,主要是看他们</p>（A）
A 对历史发展作用的大小
B 对历史发展的作用是积极的还是消极的
C 对历史发展的作用是主动的还是被动的
D 对历史发展的作用是必然的还是偶然的
解析：<p>个人按其对历史发展作用的大小，可以区分为普通个人和历史人物；按照历史人物的作用是消极的还是积极的，可以区分为正面人物和反面人物。</p>

<p>人的本质在于</p>（C）
A 人的自然属性
B 人的道德品质
C 人的社会属性
D 人的学识修养
解析：<p>人的本质不是由自然属性决定的，而是由人的社会属性决定的。马克思主义对人的本质作了界定：人的本质是一切社会关系的总和；劳动是人的本质；人的需要即人的本质。</p>

<p>在现代生产力系统中,生产工具属于</p>（A）
A 实体性因素
B 运筹性因素
C 渗透性因素
D 准备性因素
解析：<p>在现代生产力系统中,生产工具属于实体性因素。</p>

<p>在社会存在与社会意识的关系问题上, 两种根本对立的观点是</p>（C）
A 机械唯物主义和辩证唯物主义
B 可知论与不可知论
C 历史唯物主义和历史唯心主义
D 辩证法与形而上学
解析：<p>在社会存在与社会意识的关系问题上，两种根本对立的观点是历史唯物主义和历史唯心主义。世界的存在状态问题：辩证法和形而上学；哲学的基本问题第二方面—是否具有同一性：划分为可知论和不可知论。</p>

<p>党的十八大报告提出要积极培育和践行社会主义核心价值观,其中从社会层面对社会主义核心价值观基本理念的凝练表达是</p>（B）

B 自由、平等、公正、法治

D 创新、包容、开放、进取

<p>在真理问题上坚持辩证法,就是要承认任何真理都是</p>（D）

B 主观性和相对性的统一

D 绝对性和相对性的统一
解析：<p>真理只有两个特点：客观性；绝对性和相对性。</p>

<p>那种主张“天赋的才能”“天生的心灵禀赋”的观点属于</p>（A）
A 唯心主义先验论
B 旧唯物论的直观反映论
C 唯物主义反映论
D 辩证唯物论的能动反映论

<p>下列选项中, 反映实践对认识具有决定作用的是</p>（D）

<p>在意识问题上,唯心主义的错误在于</p>（C）
A 忽视意识的主观特征
B 抹煞意识的主观特征

<p>联系的观点和发展的观点是</p>（B）
A 马克思主义的理论品质

C 马克思主义的活的灵魂

<p>“盛年不重来, 一日难再晨”, 这个谚语说的是</p>（A）
A 时间具有一维性
B 物质运动具有持续性
C 时间具有有限性
D 物质运动具有顺序性
解析：<p>时间是物质运动的持续性和顺序性。所谓持续性是指任何一个事物的运动都要经历一个或长或短的过程；所谓顺序性是指事物的运动过程中不同阶段的出现有一个先后顺序关系。时间的特点是一维性，即时间总是朝着一个方向向前发展，既不是循环，也不是倒退，具有不可逆性。 比如“盛年不重来，一日难再晨”、“光阴好比河中水，只能流去不流回”等。</p>

A 人们对整个世界的根本看法和观点

C 人们对物质世界的根本看法和观点
D 人们对外部世界的根本看法和观点
解析：<p>所谓世界观，就是人们对包括自然、社会和人类思维在内的整个世界的根本看法和根本观点。</p>

<p>马克思主义是一个具有内在逻辑联系的科学体系, 其中处于核心地位的是</p>（D）
A 辩证唯物主义
B 历史唯物主义
C 政治经济学
D 科学社会主义
解析：<p>马克思主义哲学是科学的世界观和方法论，政治经济学揭示了资本主义的发展规律，处于核心地位的则是科学社会主义理论。</p>
