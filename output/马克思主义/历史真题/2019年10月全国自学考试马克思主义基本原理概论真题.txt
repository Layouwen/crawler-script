2019年10月全国自学考试马克思主义基本原理概论真题（选择题）

<p>我们实现人生价值的基础和归宿是</p>（B）
A 爱岗敬业的职业理想
B 共产主义远大理想
C 和谐共生的社会理想
D 大公无私的道德理想非选择题部分
解析：<p>坚定共产主义远大理想信念是我们实现人生价值的基础和归宿，是凝聚一切进步社会力量、推动社会不断前进的精神航标。</p>

<p>下列选项中，集中体现共产主义社会主要特征和本质要求的是</p>（A）
A 各尽所能，按需分配
B 各尽所能，按才能分配
C 各尽所能，按劳分配
D 各尽所能，按地位分配
解析：<p>马克思十分强调&quot;各尽所能，按需分配&quot;是共产主义社会的显著特征，并把它作为共产主义社会的一面旗帜加以突出论述。</p>

<p>马克思主义政党的组织原则是</p>（C）
A 集体领导制
B 个人负责制
C 民主集中制
D 民主监督制
解析：<p>民主集中制是马克思主义政党的组织原则。</p>

<p>早期空想社会主义的代表作之一《乌托邦》的作者是</p>（B）
A 法国的圣西门
B 英国的莫尔
C 英国的欧文
D 意大利的康帕内拉
解析：<p>早期空想社会主义的代表作是英国人莫尔写的《乌托邦》和意大利人康帕内拉写的《太阳城》。</p>

<p>垄断资本主义国家用多种手法剥削和控制发展中国家，其中典型的新殖民主义手法是</p>（D）
A 高价倾销发达国家生产的工业制品
B 低价收购发展中国家的资源性产品
C 以占领殖民地的方式剥削和控制发展中国家
D 打着“援助”的旗号剥削和控制发展中国家
解析：<p>当代垄断资本主义国家多是采取比较缓和隐蔽的新殖民主义手法，打着所谓援助&quot;的旗号实现其对发展中国家的剥削和控制。</p>

<p>经济全球化本质上是</p>（D）
A 生产全球化
B 贸易全球化
C 资本全球化
D 资源配置全球化
解析：<p>经济全球化本质上是资源配置的国际化。</p>

<p>当代资本主义国家中，政府对国民经济进行宏观管理和调控</p>（A）
A 能在一定程度上缓解社会矛盾
B 能够杜绝经济危机的发生
C 会使其经济发展长盛不衰
D 可以消除资本主义基本矛盾
解析：<p>在分配关系上，许多发达国家对收入分配政策进行了某些调整，如实行社会福利政策，通过再分配手段在一定程度上缓和了社会矛盾。</p>

<p>金融寡头在经济上的统治主要是通过</p>（C）
A 股份制实现的
B 有限责任制度实现的
C 参与制实现的
D 个人联合的方式实现的
解析：<p>金融寡头主要通过“参与制”实现其经济上的统治。具体地说就是金融寡头通过收买和持有一定数量股票的法实现对其它股份企业的层层控制。母公司控制子公司子公司控制孙公司形成金字塔控制结构。</p>

<p>平均利润的形成是</p>（A）
A 部门之间竞争的结果
B 部门内部竟争的结果
C 剩余价值率平均化的结果
D 资本家追逐超额利润的结果
解析：<p>不同生产部门之间平均利润的形成，是部门之间竞争的结果。</p>

<p>社会再生产的核心问题是</p>（D）
A 社会总价值的生产问题
B 社会总产品的创造问题
C 社会总价值的分配问题
D 社会总产品的实现问题
解析：<p>考察社会总资本的运动即考察社会资本再生产，核心问题是要分析在社会资本再生产过程中，社会总产品的各个构成部分是如何实现的。</p>

<p>相对剩余价值的获得是</p>（D）
A 延长工作日、增加剩余劳动时间的结果
B 工作日不变、增加必要劳动时间的结果
C 个别企业提高劳动生产率的结果
D 全社会劳动生产率普遍提高的结果
解析：<p>生产部门的劳动生产率普遍提高即整个社会劳动生产率提高后，单位商品价值便会下降，劳动力价值随之下降，再生产劳动力价值的必要劳动时间便会缩短，剩余的劳动时间则相应延长，从而生产出相对剩余价值。</p>

<p>商品经济是</p>（C）
A 为他人而生产的经济形式
B 存在于一切社会的经济形式
C 直接以交换为目的的经济形式
D 以自给自足为特征的经济形式
解析：<p>商品生产和商品交换为内容，直接以交换为目的而进行生产的经济形式就是商品经济。</p>

<p>下列各项属于技术社会形态序列的是</p>（D）
A 自然经济社会
B 商品经济社会
C 产品经济社会
D 信息社会
解析：<p>在电子时代，信息技术和信息产业在技术体系和产业结构中占了主导地位，因而可以称之为信息社会。</p>

<p>生产力和生产关系的矛盾、经济基础和上层建筑的矛盾是人类社会的两对基本矛盾，它们之间是</p>（C）
A 相互决定的关系、无主次之分
B 相互平行的关系
C 相互制约的关系、有主次之分
D 相互孤立的关系
解析：<p>生产力和生产关系之间的矛盾、经济基础和上层建筑之间的矛盾这两对基本矛盾，不是互相孤立、互相平行的，而是相互制约、有主次之分的。</p>

<p>在劳动资料方面，最能标志生产力发展水平的因素是</p>（B）
A 信息传递系统
B 生产工具
C 自动控制系统
D 动力系统
解析：<p>生产工具是劳动资料的主要内容，是生产力发展水平的主要标志。</p>

<p>地理环境在社会发展中的作用主要通过对个体意识的影响实现</p>（B）
A 对个体意识的影响实现
B 对物质生产的影响实现
C 对群体意识的影响实现
D 对公序良俗的影响实现
解析：<p>地理环境通过对生产的影响，加速或延缓社会的发展。</p>

<p>坚持马克思主义哲学的认识路线，就必须坚持</p>（A）
A 一切从客观实际出发
B 一切从主观愿望出发
C 一切从个人需要出发
D 一切从集体意志出发
解析：<p>坚持马克思主义哲学，坚持马克思主义哲学的认识路线，就必须坚持一切从实际出发。</p>

<p>下列各项正确说明真理问题的是</p>（D）
A 真理是因人而异的
B 有用的就是真理
C 真理是凝固不变的
D 真理是有价值的
解析：<p>真理能够指导社会实践，人们通过在真理指导下的实践，成功地改造世界，以这种实践的成果来满足人的需要，这就是真理所表现出来的价值属性。</p>

<p>辩证唯物主义认为，认识的本质是</p>（C）
A 主体对客体的自由选择
B 主体对客体的自由创造
C 主体对客体的能动反映
D 主体对客体的科学评价
解析：<p>认识的本质是主体对客体的能动反映。</p>

<p>意识是人脑的机能，这说的是</p>（C）
A 人脑是意识的唯一源泉
B 意识不具有客观性
C 人脑是意识的物质器官
D 意识不具有主观性
解析：<p>意识是人脑的机能，人脑是意识的物质器官。人类意识高于动物心理，主要在于它是以人脑为物质基础的。</p>

<p>自觉能动性是实践的一个基本特点，这表明实践是</p>（B）
A 主体纯粹的思维活动
B 主体有意识、有目的的活动
C 主体感性的物质活动
D 主体社会性、历史性的活动
解析：<p>实践是主体有意识、有目的的活动。</p>

<p>与自然规律相比，社会规律是</p>（A）
A 通过人的活动实现的
B 不能正确认识的
C 盲目自发地起作用的
D 可以人为改变的
解析：<p>社会发展规律是通过人们的自觉活动实现的。</p>

<p>马克思主义认为，社会生活在本质上是</p>（A）
A 实践的
B 多样的
C 可知的
D 变化的
解析：<p>马克思认为社会生活在本质上是实践的。</p>

<p>理论联系实际是学习马克思主义的根本方法，这是由马克思主义的</p>（C）
A 科学性决定的
B 革命性决定的
C 实践性决定的
D 阶级性决定的
解析：<p>理论联系实际是学习马克思主义的根本方法。首先，这是由马克思主义的实践性决定的。</p>

<p>在马克思主义科学体系中，作为世界观和方法论的是</p>（A）
A 马克思主义哲学
B 马克思主义政治学
C 马克思主义政治经济学
D 科学社会主义理论
解析：<p>马克思主义的各个组成部分不是彼此孤立、互不联系的，而是组成一个具有内在逻辑联系的科学体系，其中马克思主义哲学是科学的世界观和方法论，政治经济学揭示了资本主义的发展规律，处于核心地位的则是科学社会主义理论。</p>
