马克主义基本原理概论第二章节练习题（选择题）

党的思想路线的核心是（C）
A 一切从实际出发
B 理论联系实际
C 实事求是
D 实践是检验真理的唯一标准
解析：党的思想路线的核心  C P125

认识的主体和认识的客体是一对矛盾的两个方面，它们之间的关系是（C）
A 具体与抽象的关系
B 主观与客观的关系
C 对立又统一的关系
D 绝对与相对的关系
解析：认识的主体和认识的客体关系&nbsp;C P107

辩证唯物主义认识论首要的基本观点是（D）
A 可知论的观点
B 唯物论的观点
C 辩证法的观点
D 实践的观点
解析：辩证唯物主义认识论 D P102

“没有理性，眼睛是最坏的见证人”。这句话强调的是（A）
A 仅同事物的现象相符合的不一定是真理
B 只要同事物的相符合的就是真理
C 只要观察事物的外部联系就能获得真理
D 只要观察事物的偶然联系就能获得真理
解析：理性的认识 A P110

真理和谬误在一定条件下可相互转化，这是（A）
A 辩证唯物主义的观点
B 旧唯物主义的观点
C 诡辩论的观点
D 经验论的观点
解析：真理和谬误 A P 116

人们认识世界的目的在于（D）
A 发展个人的兴趣爱好
B 实现对自身能力的欲望
C 满足人类的求知欲望
D 实现对客观改造
解析：认识的目的 D P104

在检验真理的标准问题上，正确的说法是（B）
A 以往的理论是检验真理的标准
B 社会实践是检验真理的标准
C 统治者的意志是检验真理的标准
D 多数人的意见是检验真理的标准
解析：检验真理的标准问题 B P118

认识的真正本质是（C）
A 主体对客体改造与被改造的疯关系
B 主体对客体满足和被满足的关系
C 主体对客体反映与被反映的关系
D 主体和客体相互依存和作用的关系
解析：认识的本质  C P108

理性认识是对客观事物的（A）
A 全体、本质和内部反映
B 现象的直接反映
C 生动、具体的外部形象的反映
D 外部联系的片面反映
解析：理性认识的含义 A P110

判断一种认识是不是真理，要看它（D）
A 能否被多数人认同
B 能否付诸实际行动
C 能否满足人的需要
D 能否在实践中达到预期效果
解析：认识的作用 D P118-119

一个完整的认识过程，需要两次飞跃。下列选项中属于第二次飞跃的是（A）
A 理论付诸实践
B 分析实际情况
C 总结经验教训
D 进行社会调查
解析：认识过程第二次飞跃 A P112

实践作为检验真理的标准，其不确定性的是由于（B）
A 不同的人、不同的阶段各有其不同的实践标准
B 实践对真理的检验具有反复性
C 任何实践检验都需要逻辑证明作为补充的手段
D 有些真理无法通过实践来加以检验

绝对真理与相对真理是（D）
A 两种完全相同的真理
B 丙个有着固定不变的界限的真理
C 两种完全不同的真理
D 同一个真理的两种不同属性和方面
解析：绝对真理与相对真理 D P118

C 实践—认识—实践
