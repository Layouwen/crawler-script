10、清朝统治集团中倡导洋务的首领是（C）
A 曾国藩
B 左宗棠
C 奕诉
D 李鸿章
解析：本题考查的是重要历史人物。19世纪60年代，西方列强逐步加深了对中国的侵略。封建统治阶级中的部分成员如奕诉、曾国藩、李鸿章、左宗棠、张之洞，刘坤一等，主张仿造西方的武器装备，学习西方的科学技术，以“自强”“求富”为目标，兴办洋务。这些官员被称为洋务派。恭亲王爽诉是清朝统治集团中倡导洋务的首领

11、对洋务派兴办洋务事业的指导思想最先做出比较完整表述的是（D）
A 奕诉
B 张之洞
C 刘坤一
D 冯桂芬
解析：本题同样是考查历史人物的作为。冯桂芬对洋务派兴办洋务事业的指导思想最先做出比较完整的表述。他说，以中国之伦常名教为原本，辅以诸国富强之术。这个思想后来被进一步概括为“中学为体，西学为用"。

12、洋务派举办近代军事工业的首要目的是（B）

B 镇压太平天国起义
C 发展工业，促进对外贸易
D 为民用企业的创办打下基础
解析：本题考查的是洋务派举办近代军事工业的首要目的。洋务派举办洋务是从近代军用工业着手的，其首要目的是镇压太平天国起义；同时，也有加强海防、边防抵御外国侵略的目的在内。

13、到19世纪90年代洋务派建成的新式海军中的主力是（A）
A 北洋水师
B 广东水师
C 南洋水师
D 福建水师
解析：本题考查洋务派建成的新式海军中的主力。1874年，日本派兵侵略我国台湾，清政府兴起海防之议。到19世纪90年代，清政府分别建成福建水师、广东水师、南洋水师、北洋水师。其中，北洋水师是清政府海军的主力，一直由李鸿章管辖。

18、戊戌维新时期，维新派在上海创办的影响较大的报刊是（A）
A 《时务报》
B 《国闻报》
C 《湘报》
D 《新民丛报》
解析：本题考查维新派创建的报刊。維新派创办的影响较大的报刊是梁启超主笔的上海《时务报》，还有严复主办的天津《国闻报》及湖南的《湘报》。《新民丛报》在1902年2月由梁启超创办于日本横滨，是20世纪初资产阶级改良派的重要刊物。

19、实质上是资产阶级思想与封建主义思想在中国的第一次正面交锋的是（A）
A 维新派和守旧派的论战

C 改革派与保守派的论战
D 洋务派与保守派的论战
