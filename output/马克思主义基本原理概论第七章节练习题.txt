马克思主义基本原理概论第七章节练习题（选择题）

实现了人的“自由个性”的发展是（C）
A 社会主义社会之中的人的生存状态
B 资本主义社会之中的人的生存状态
C 共产主义社会之中的人的生存状态
D 资本主义社会以前的人的生存状态
解析：人的“自由个性”的发展 C P278

共产主义社会的两个阶段依次是（D）
A 共产主义社会、社会主义社会
B 自由发展的社会，民主社会
C 民主社会、自由发展的社会
D 社会主义社会、共产主义社会
解析：共产主义社会的两个阶段 D P273

集中体现着共产主义社会主要特征和本质 要求的原则标志是（A）
A 各尽所能，按需分配
B 社会生产力高度发展
C 精神境界极大提高
D 阶级的消灭和国家自行消亡
解析：集中体现着共产主义社会主要特征和本质 要求的原则标志 A P275-276

------开创了我国社会主义发展的崭新道路（B）
A 社会主义和谐社会
B 建设中国特色社会主义
C 彻底消灭剥削、消灭两极分化
D 实现全体人民共同富裕
解析：建设中国特色主义 B P289

<p>恩格斯指出，国家不是“被废除”的，而是</p>（D）
A 统治阶级自动废除的
B 突然消亡的
C 被统治阶级推翻
D 自行消亡的
解析：<p>恩格斯指出，国家不是“被废除”的，而是国家自行消亡的。</p>

共产主义社会的分配原则是（D）
A 按生产要素分配
B 按劳分配
C 平均分配
D 按需分配
解析：按需分配 D P275

我国正处在社会主义初级阶段，人民的历史使命和共同理想是（A）
A 建设中国特色社会主义
B 实现共产主义
C 建设社会主义市场经济
D 实现共同富裕
解析：我国社会主义初级阶段人们的历史使命和共同理想 A P289

共产主义社会的生产资料所有制是（A）
A 生产资料的单一社会公有制
B 多种形式的生产资料公有制并存
C 公有制和非公有制经济形式并存
D 生产资料的个人和公共所有并存
解析：共产主义社会的生产资料所有制 A P276

社会阶级的消灭是以-----的阶段为前提的（C）
A 生产资料完全公有制
B 社会化大生产
C 生产高度发展
D 民主政治高度发达
解析：阶级的消灭 C P277

A <p>社会生产力的高度发展</p>
B <p>人与人关系的高度和谐</p>
C <p>人的思想觉悟的极大提高</p>
D <p>自然生态环境的极大改善</p>
解析：<p>实现共产主义社会的根本条件和基础是：社会生产力的高度发展。任何一种社会制度，都必须建立在一定的物质基础之上，即必须具备一定的物质条件和技术条件，这就是该社会的生产力。共产主义社会是人类社会发展的最高社会形态，它的物质基础,是远远高于以往一切社会的高度发达的社会生产力。社会生产力是社会发展和社会进步的最终决定力量，是全部社会历史的物质基础。</p><p><br/></p>

走向共产主义社会的必经阶段是（C）
A 民主主义社会
B 社会主义初级阶段
C 社会主义社会
D 社会主义过渡时期
解析：走向共产主义社会的必经阶段 C P286

社会主义和共产主义共有的经济关系是（B）
A 实行按需分配原则
B 实行生产资料的社会公有制
C 实行按劳分配原则
D 实行商品经济、市场经济体制
解析：社会主义和共产主义共有的经济关系 B P281

----是人们对社会制度和社会面貌的预见和期望（C）
A 职业理想
B 共产主义理想
C 社会理想
D 生活理想
解析：社会理想 C P288

我们实现人价值的基础和归宿是（B）
A 社会主义远大理想
B 共产主义远大理想
C 人类社会的远大理想
D 无产阶级远大理想
解析：我们实现人价值的基础和归宿 B P289
