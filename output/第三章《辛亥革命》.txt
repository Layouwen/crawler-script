1、1904年至1905年，为争夺侵略权益公然在中国东北进行战争的是（D）
A 美国与俄国
B 美国与英国
C 英国与日本
D 日本与俄国
解析：本题考查日本与俄国对中国的侵略。1904-1905年，日、俄两国为争夺在华利益竟然在中国东北进行战争，清政府却宣布“局外中立”，结果日本战胜俄国，俄国将所攫取的中国东北南部所有一切权益“转让”给日本

4、中国第一个资产阶级性质的政党是（B）
A 兴中会
B 中国同盟会
C 光复会
D 中国国民党
解析：本题主要考查1905年成立的中国同盟会的性质。之前成立的兴中会、华兴会、光复会等都是资产阶级革命团体；中国同盟会则是近代中国第一个全国性的资产阶级性质的政党，它的成立标志着中国资产阶级民主革命进入了一个新阶段。

5、中国同盟会的机关报是（B）
A 《新青年》
B 《民报》
C 《湘江评论》
D 《新民丛报》
解析：本题主要考查资产阶级革命浓创办的报刊。中国同盟会成立后，就创办《民报》作为机关报。《新民丛报》是当时资产阶级改良派的主要舆论阵地；《新青年》《湘江评论》是辛亥革命后创办的报刊

6、下列关于“民族主义“说法不正确的是（D）

12、1911年4月27日，在黄兴的亲自带领下，120多名革命志士在广州举行起义，史称（D）
A 辛亥革命
B 萍浏醴起义
C 秋收起义
D 黄花岗起义
解析：本题考查黄花岗起义。同盟会成立以后，以孙中山为首的资产阶级革命派，把开展武装起义放在了首要地位，组织了多次武装起义。其中影响最大的是1911年的广州起义。1911年4月27日，在黄兴的亲自带领下，120多名革命志士在广州举行起义。起义失败后，遇难者有72人被葬于广州红花岗（改名为黄花岗），史称黄花岗起义

13、中华民国南京临时政府的性质是（A）
A 资产阶级共和国性质的革命政权
B 大地主大资产阶级专制的政府
C 无产阶级占主导地位的政权
D 各阶级联合执政的政府
解析：本题主要考查武昌起义后成立的中华民国南京临时政府的性质。由于在人员构成上，资产阶级革命派控制着这个政权；在实行的各项政策措施上，集中体现了中国民族资产阶级的愿望和利益，也一定程度上符合广大中国人民的利益；因此，中华民国南京临时政府是一个资产阶级共和国性质的革命政权。

14、中国历史上第一部具有资产阶级共和国宪法性质的法典是（A）
A 《中华民国临时约法》
B 《钦定宪法大纲》
C 《中华民国约法》
D 《中华民国宪法》
解析：本题考查中国历史上第一部具有资产阶级共和国宪法性质的法典。1912年，临时参议院颁布《中华民国临时约法》。这是中国历史上第一部具有资产阶级共和国宪法性质的法典。它以根本大法的形式废除了两千年来的封建君主专制制度，确认了资产阶级共和国的政治制度

15、辛亥革命的性质是（B）
A 无产阶级社会主义革命
B 旧式的资产阶级民主主义革命
C 新式的资产阶级民主主义革命
D 资产阶级维新运动
解析：本题主要考查辛亥革命的性质。这道题有一定难度。在中国，资产阶级民主主义革命分为旧式的资产阶级民主主义革命和新式的资产阶级民主主义革命，辛亥革命是由资产阶级领导的，因此是旧式的资产阶级民主主义革命，而不是新式的资产阶级民主主义革命，更不是无产阶级社会主义革命

16、辛亥革命后，窃夺革命果实的是（C）
A 黎元洪
B 李鸿章
C 袁世凯
D 段祺瑞
解析：本题主要考查的是历史人物的作为。辛亥革命后，袁世凯窃夺了辛亥革命的果实，建立了代表大地主和买办资产阶级利益的北洋军阀政权，开始了北洋军阀统治中国的时期

17、袁世凯建立的北洋军阀政权代表的是（A）
A 大地主和买办资产阶级的利益
B 大资产阶级的利益
C 民族资本家和中小资产阶级的利益
D 无产阶级和广大人民的利益
解析：本题主要考查袁世凯建立的北洋军阀政权的阶级性质。袁世凯建立的北洋军阀政权集中代表大地主和买办资产阶级的利益，实行军阀官僚的专制统治。
